This is a glossary of terms used within the ZackiSchool WorldWideWeb project. In most cases, their use corresponds to conventional use in hypertext circles.

Glossary of Hypertext Terms

Anchor
An area within the content of a node which is the source or destination of a link. Clicking the anchor follows the link, displaying the content at the opposite end.
Annotation
Linking a new commentary node to an existing node. Annotation and multiuser collaboration
Authoring
The process of writing a document.
Back link
A link implied from an existing link in the opposite direction. Building back-links.
Browser
A program for reading hypertext.
Button
An anchor that is the source of a link.
Card
An alternative term for a node.
Client
A program requesting services from a server.
Database
A collection of nodes.
Daemon
A program running independently, performing tasks like back-links.
Document
A node or collection of nodes.
Domain
A unit of protection.
External
A link to a node in another database.
Node
A unit of information.
Link
A relationship between two anchors.
Server
A program providing services to a client.
Web
A set of interconnected nodes.
